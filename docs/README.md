| **[Migration from CocoaPods to SwiftPM](MIGRATION_FROM_COCOAPODS_TO_SPM.md)** | Step-by-step guide to migrate from the CocoaPods Gradle plugin to SwiftPM and the Kotlin/Native direct Xcode integration. Use this when moving an existing KMP + CocoaPods iOS setup to SwiftPM.                                                                            |
